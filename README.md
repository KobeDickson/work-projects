## Adding files or folders

To add a folder locally and push it to this repo:

1. Create the folder and add a placeholder file (Git doesn't track empty folders):
   mkdir -p path/to/folder
   echo "placeholder" > path/to/folder/.gitkeep
2. Stage, commit and push:
   git add path/to/folder/.gitkeep
   git commit -m "Add path/to/folder with .gitkeep"
   git push origin main
